class facturaTableSQL.ModeloTaboa(datos)[fuente]¶

Bases: QAbstractTableModel

Clase que define un modelo de tabla para datos.

Este modelo de tabla se utiliza para representar datos en una interfaz de usuario utilizando el framework Qt. Hereda de la clase QAbstractTableModel.

columnCount(index)[fuente]¶

Devuelve el número de columnas en el modelo de tabla.

Si hay datos presentes, devuelve la longitud de la primera fila de datos. Si no hay datos, devuelve 0.

Parámetros:

index (int) – El índice de la columna.

Devuelve:

El número de columnas en el modelo de tabla.

Tipo del valor devuelto:

int

data(index, role=ItemDataRole.DisplayRole)[fuente]¶

Devuelve los datos en el índice especificado para un rol dado.

Si el índice es válido y el rol es DisplayRole o EditRole, se devuelve el valor de los datos en el índice como una cadena.

Parámetros:

  • index (QModelIndex) – El índice para el que se solicitan los datos.

  • role (Qt.ItemDataRole) – El rol para el que se solicitan los datos (por defecto: DisplayRole).

Devuelve:

Los datos en el índice especificado para el rol dado, o None si el índice no es válido.

Tipo del valor devuelto:

str or None

setData(index, value, role)[fuente]¶

Establece los datos en el índice especificado con el valor dado para el rol dado.

Si el rol es EditRole, actualiza los datos en el índice con el nuevo valor proporcionado.

Parámetros:

  • index (QModelIndex) – El índice en el que se establecerán los datos.

  • value (str) – El nuevo valor para establecer en el índice.

  • role (Qt.ItemDataRole) – El rol para el que se establecerán los datos.

Devuelve:

True si se establecieron los datos correctamente, False en caso contrario.

Tipo del valor devuelto:

bool
